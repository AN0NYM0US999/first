What you need to learn to become a Web Developer

To become a web developer you have to learn some technologies. If you have no experience you can start with the following list-

  1. HTML
  2. CSS
  3. JavaScript
  4. PHP
  5. MySQL

You have to learn HTML to structure a simplest web page. It is the base of the remaining. CSS is used to stylize the page. JavaScript is used to provide a great user experience.
